# WG Holdings Website Migration Project

## Project Overview
This project is a complete migration of wgholdings.co.nz to a modern HTML, CSS, and JavaScript structure with a focus on SEO optimization and performance.

## Project Structure

```
wgholdings-website/
├── index.html              # Homepage
├── assets/
│   ├── css/               # Stylesheets
│   │   ├── main.css       # Main stylesheet
│   │   ├── responsive.css # Responsive design
│   │   └── components.css # Reusable components
│   ├── js/                # JavaScript files
│   │   ├── main.js        # Main JavaScript
│   │   └── utils.js       # Utility functions
│   ├── images/            # Image assets
│   │   ├── logo/
│   │   ├── pages/
│   │   └── icons/
│   └── fonts/             # Custom fonts (if any)
├── pages/                 # Additional HTML pages
├── sitemap.xml            # SEO sitemap
├── robots.txt             # Search engine directives
└── README.md              # This file
```

## Development Workflow

### Page-by-Page Migration Process:

1. **Audit Current Page**
   - Take screenshots
   - Document all content
   - Note all interactive elements
   - Record SEO metadata

2. **Build HTML Structure**
   - Use semantic HTML5 tags
   - Implement proper heading hierarchy
   - Add alt text for images
   - Include meta tags

3. **Style with CSS**
   - Match original design
   - Ensure responsive behavior
   - Optimize for performance

4. **Add JavaScript**
   - Implement interactivity
   - Add form validation
   - Optimize user experience

5. **SEO Optimization**
   - Verify meta tags
   - Test mobile responsiveness
   - Check page speed
   - Validate HTML

6. **Testing**
   - Cross-browser testing
   - Mobile device testing
   - Accessibility testing
   - Performance testing

## Getting Started

1. Place all images in `assets/images/` folder
2. Update content in HTML files
3. Customize colors in `assets/css/main.css`
4. Test locally by opening `index.html` in a browser
5. Deploy to your hosting provider

## Deployment

Before deploying:
1. Test all pages thoroughly
2. Validate HTML (https://validator.w3.org/)
3. Check mobile responsiveness
4. Run PageSpeed Insights
5. Verify all links work
6. Submit sitemap to Google Search Console
