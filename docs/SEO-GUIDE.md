# WG Holdings - SEO Checklist & Guidelines

## On-Page SEO Elements

### Essential Meta Tags (Every Page Must Have):
- [ ] `<title>` tag (50-60 characters, unique per page)
- [ ] Meta description (150-160 characters, compelling)
- [ ] Meta viewport for mobile responsiveness
- [ ] Canonical URL tag
- [ ] Open Graph tags for social sharing
- [ ] Twitter Card tags
- [ ] Language declaration (lang="en")
- [ ] Character encoding (UTF-8)

### Content SEO:
- [ ] One H1 tag per page (primary keyword)
- [ ] H2-H6 tags in logical hierarchy
- [ ] Alt text for all images (descriptive, keyword-rich)
- [ ] Internal linking strategy
- [ ] External links open in new tab (rel="noopener")
- [ ] Keyword density 1-2% (natural usage)
- [ ] Content length: minimum 300 words per page
- [ ] Regular content updates

### Technical SEO:
- [ ] Clean, semantic HTML5 structure
- [ ] Schema.org structured data
- [ ] XML sitemap
- [ ] robots.txt file
- [ ] Favicon and app icons
- [ ] SSL certificate (HTTPS)
- [ ] 404 error page
- [ ] Fast loading speed (<3 seconds)
- [ ] Mobile-friendly design
- [ ] Responsive images with srcset
- [ ] Lazy loading for images
- [ ] Minified CSS and JavaScript
- [ ] Gzip compression enabled
- [ ] Browser caching configured

### URL Structure:
✅ Good: `wgholdings.co.nz/about-us`
❌ Bad: `wgholdings.co.nz/page?id=123`

- [ ] Short, descriptive URLs
- [ ] Use hyphens, not underscores
- [ ] Lowercase only
- [ ] No special characters
- [ ] Include target keywords

## Image Optimization

### Best Practices:
- [ ] Use WebP format with JPG/PNG fallback
- [ ] Compress images (TinyPNG, ImageOptim)
- [ ] Descriptive file names (service-consultation.jpg)
- [ ] Alt text for all images
- [ ] Responsive images with srcset
- [ ] Lazy loading enabled
- [ ] Maximum file size: 200KB
- [ ] Dimensions appropriate for usage

### Recommended Sizes:
- Hero images: 1920x1080px
- Feature images: 800x600px
- Thumbnails: 400x300px
- Logo: 200x50px (transparent PNG)
- Icons: 64x64px or SVG

## Performance Optimization

### Page Speed Targets:
- [ ] First Contentful Paint: < 1.8s
- [ ] Largest Contentful Paint: < 2.5s
- [ ] Time to Interactive: < 3.8s
- [ ] Total Blocking Time: < 200ms
- [ ] Cumulative Layout Shift: < 0.1

### Optimization Techniques:
- [ ] Minify HTML, CSS, JavaScript
- [ ] Enable compression (Gzip/Brotli)
- [ ] Optimize images
- [ ] Use CDN for static assets
- [ ] Implement browser caching
- [ ] Remove unused CSS/JavaScript
- [ ] Defer non-critical JavaScript
- [ ] Use async loading where possible
- [ ] Preload critical resources
- [ ] Reduce redirects

### Internal Linking:
- Link related pages together
- Use descriptive anchor text
- Create a logical site structure

### External Links:
- Link to authoritative sources

- Check for broken links regularly
- Quality over quantity
